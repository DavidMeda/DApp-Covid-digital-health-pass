Covid Digital Health Pass

Add permission Ministry of Health


This section provide to add MINISTRY of Health to administer the autorizated Hubs.
Only owner of contract is allowed to add new Ministry.
It's necessary to use a unique address for each Ministry.

Add permission Hub medical


This section provide to add HUB that are autorizated to test and vacinate people against COVID-19.
Only Ministry is allowed to add new Hub.
It's necessary to use a unique address for each Hub.

Remove permission Ministry of Health


This section provide to remove Ministry of Health from medical authorities.
Only owner of contract is allowed to remove a Ministry.
It's necessary to use a unique address of the Ministry.

Remove permission medical Hub


This section provide to remove HUB that are autorizated to test and vacinate people against COVID-19.
Only Ministry is allowed to remove a Hub.
It's necessary to use a unique address for each Hub.

Antibody Test certificate publication


This section provide to publish a COVID test result of people to blockchain. For privacy preserving no sensible data is store to blockchain, only hash of documents is stored.
It's necessary to identify test-taker with document ID and provide:
  1. Document ID to calculate unique hash
  2. COVID test result to calculate unique hash
  3. Posivity result of test
  4. Account address of test-taker
It's possibly to upload Test result document to IPFS (distribuited storage) and get the link. Only autorizated Hub is allowed to submit Test result.

Vaccine certificate publication


This section provide to publish a COVID vaccine certicate of person to blockchain. For privacy preserving no sensible data is stored into blockchain, only hash of documents is stored.
It's necessary to identify person who gets vaccinated with document ID and provide:
  1. Document ID to calculate unique hash
  2. COVID vaccine certicate to calculate unique hash
  3. Account address of person
It's possibly to upload COVID vaccine ceriticate to IPFS (distribuited storage) and get the link. Only autorizated Hub is allowed to submit vaccine certificate.

Antibody test result verification


This section provide to verificate antibody test result.
It's necessary to identify person with document ID and provide:

  1. Document ID to calculate unique hash
  2. COVID test result or download it from IPFS to calculate unique hash
  3. Account address of test-taker

Vaccine certificate verification


This section provide to verificate vaccine certificate.
It's necessary to identify person with document ID and provide:

  1. Document ID to calculate unique hash
  2. COVID vaccine certificate or download it from IPFS to calculate unique hash
  3. Account address of user
